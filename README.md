# sellsniper.com

> The AI sales agent. Finds the exact humans on Earth who would love your work.

## What it is

You feed sellsniper any artifact: a tool, a landing page, a tweet, a deck, a song, an app, a domain. It finds the exact humans on Earth who would LOVE to see it. Then it drafts the message that gets their attention. Stop shouting into the void. Ship like a sniper.

This is the distribution layer for the [Aether](https://adampang.com) portfolio. Every product Adam ships gets distributed through sellsniper.

## Live

https://sellsniper.com

## Stack

- Next.js (App Router) + TypeScript + Tailwind CSS
- Vercel deployment + PostHog analytics

## Build

```bash
npm install
npm run dev
```
